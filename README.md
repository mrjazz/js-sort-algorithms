js-sort-algorithms
==================

# Implementation of different sort algorithms in javascript

This project include a few implementations of sort algorithms:

* shellSort(Sedgewick)
* shellSort(Fibo)
* shellSort(Hibbard)
* shellSort(Pratt)
* insertionSort
* mergeSort
* quickSort
* selectionSort
* combSort
* heapSort
* cocktailSort
* quick3Sort
* binaryTreeSort


Also it include nice vsualizer that allow you track algorithm logic step by step.
